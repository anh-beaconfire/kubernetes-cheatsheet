Configuration

ConfigMap

  • ConfigMap stores non-sensitive configuration.
  • do not store secrets in ConfigMaps
  • use versioned config changes
  • restart pods when config changes if app does not reload dynamically
apiVersion: v1
kind: ConfigMap
metadata:
  name: api-config
data:
  LOG_LEVEL: "info"
  FEATURE_FLAG: "true"

---

envFrom:
  - configMapRef:
      name: api-config
