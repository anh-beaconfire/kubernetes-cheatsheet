Architecture

Declarative Model and Reconciliation

  • Declarative model: you define the desired state
  • Actual state: what is currently running in the cluster
  • Controllers detect differences between desired and actual
  • Reconciliation loop continuously drives the system toward desired state
  • This enables self‑healing, automation, and drift correction
