Networking

Service

  • A Service provides stable networking and load balancing for a set of pods.
  • ClusterIP
  • NodePort
  • LoadBalancer
  • ExternalName
  • use ClusterIP for internal services
apiVersion: v1
kind: Service
metadata:
  name: api-service
spec:
  type: ClusterIP
  selector:
    app: api
  ports:
    - port: 80
      targetPort: 8080
