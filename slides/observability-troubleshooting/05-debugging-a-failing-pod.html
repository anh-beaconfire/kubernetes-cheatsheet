Troubleshooting

Debugging a Failing Pod

  • Check Pod status — see if it's Pending, CrashLoopBackOff, ImagePullBackOff, or OOMKilled. Status gives the first clue.
  • Describe the Pod — inspect Conditions, container states, Events, restart count, probe failures, and image pull issues.
  • Check logs — look for stack traces, startup failures, missing configs, connection errors, or crashes.
  • Check Events — identify scheduling failures, mount errors, node pressure, probe failures, or eviction warnings.
  • Exec into Pod (if running) — verify env vars, config files, DNS, network connectivity, permissions, and dependencies.
  • Check Deployment/ReplicaSet — confirm rollout status, unavailable replicas, wrong image tags, or HPA scaling issues.
# 1. Check Pod status (Pending, CrashLoopBackOff, OOMKilled)
kubectl get pods -o wide

# 2. Describe Pod (Conditions, Events, container state)
kubectl describe pod 

# 3. Check logs (crashes, errors, startup failures)
kubectl logs  -c 
kubectl logs  --previous   # last crash

# 4. Check cluster Events (scheduling, mounts, pressure)
kubectl get events --sort-by=.metadata.creationTimestamp

# 5. Exec into Pod (inspect env, config, DNS, network)
kubectl exec -it  -- sh

# 6. Check Deployment/ReplicaSet (rollout issues)
kubectl describe deployment 
kubectl describe rs
Issue Common Reason Common Fix
Pod Pending Scheduling constraints or node pressure Check Events, adjust resource requests, fix taints/affinity
CrashLoopBackOff App crash, bad config, missing dependency Fix config/env vars, dependencies, startup logic
ImagePullBackOff Wrong tag or private registry auth failure Use correct tag, fix registry creds, verify image exists
OOMKilled Memory limit too low or memory leak Increase memory limit, fix leak, tune app usage
Probe Failures App not ready or wrong probe config Fix probe path/port, increase startup time, adjust thresholds
FailedMount PVC not bound or wrong StorageClass Bind PVC, correct StorageClass, fix volume permissions
Unavailable Replicas Rollout failure or HPA scaling issues Fix rollout, correct image, adjust HPA targets
