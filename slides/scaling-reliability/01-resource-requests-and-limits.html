Scaling

Resource Requests and Limits

  • requests influence scheduling
  • limits cap resource usage
  • missing requests causes poor scheduling
  • wrong limits can cause throttling or OOMKilled
  • always set requests
  • set memory limits carefully
resources:
  requests:
    cpu: "250m"
    memory: "256Mi"
  limits:
    cpu: "500m"
    memory: "512Mi"
